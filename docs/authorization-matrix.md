# Authorization Matrix

Structural management permissions:

- `structural-offices.manage`
- `structural-offices.view`
- `structural-offices.assignUsers`
- `structural-offices.manageEmployees`

Relationship permissions:

- `relationships.viewAny`
- `relationships.view`
- `relationships.create`
- `relationships.update`
- `relationships.delete`
- `relationships.restore`

Functional reporting permissions:

- `functional-reporting.view`
- `functional-reporting.viewReports`
- `functional-reporting.supervise`
- `functional-reporting.approveRequests`
- `functional-reporting.exportReports`

Functional reporting is intentionally narrower than structural management.
